# C++ LeetCode Flashcards for Data Structures & Algorithms Interview Preparation

This [website](https://yao-jason.github.io/cpp-leetcode-flashcards/) offers a curated collection of C++ code flashcards specifically designed to solve LeetCode problems for data structures & algorithms interview preparation.

## 🚀 Key Philosophy
Unlike traditional "code banks" that list hundreds of problems, this tool focuses on **Pattern Recognition**:
- **Triggers**: Identifying keywords in a problem that signal a specific algorithm.
- **Minimalist Templates**: The most efficient C++ core logic stripped of problem-specific noise.
- **Gotchas**: Direct reminders of common C++ pitfalls like integer overflow, boundary conditions, and memory management.

## 🛠️ Tech Stack
- **Frontend**: Vanilla JS, CSS Flexbox
- **Highlighting**: [Highlight.js](https://highlightjs.org/) (GitHub Dark Theme)
- **Data**: Centralized `data.json` for easy pattern updates

## 📂 How to Use
1. Select a topic from the sidebar.
2. Review the **Triggers** to build mental associations.
3. Memorize the **Core Template**.
4. Check the **Gotchas** to avoid common interview mistakes.
